## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| internet -> FastAPI (GET /jobs/{id}) | Cliente nao autenticado pode pollar status; sem rate limit, eh DoS vector |
| internet -> FastAPI (GET /files/{id}) | Cliente nao autenticado pode baixar WAV; sem rate limit, eh DoS vector pesado |
| internet -> FastAPI (GET /health) | Cliente nao autenticado expoe estado do Redis; intencional (necessario para monitoring) |
| FastAPI -> Redis (limiter backend) | _real_ip usa request.client.host (TCP, nao spoofavel via header) |
| filesystem (multi-user) -> /tmp/sg_*.wav | Outros usuarios do sistema poderiam ler WAVs sem chmod 0o600 |
| filesystem (multi-user) -> ./start.sh | Outros usuarios poderiam executar/modificar start.sh sem chmod 750 |
