## Trust Boundaries

| Boundary | Descrição |
|----------|-----------|
| Moisés → notebook (SSH/Tailscale) | Operador se conecta ao notebook via rede Tailscale privada — conexão autenticada |
| notebook → repositório git | git pull e git push requerem credenciais do Moisés |
| outputs → 12-SETUP-LOG.md | Outputs do sistema copiados para arquivo de log — sem dados sensíveis esperados |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-08 | Information Disclosure | 12-SETUP-LOG.md commitado no repo | accept | Log contém apenas outputs de comandos de sistema — sem senhas, tokens ou dados pessoais |
| T-12-09 | Tampering | git pull puxando código modificado antes da execução | accept | Moisés verifica o conteúdo do script antes de executar com sudo — padrão de operação normal |
| T-12-10 | Denial of Service | Reboot interrompe sessão SSH ativa | accept | Reboot é intencional e esperado; operador sabe que vai reconectar — instrução explícita no passo 3 |
</threat_model>
