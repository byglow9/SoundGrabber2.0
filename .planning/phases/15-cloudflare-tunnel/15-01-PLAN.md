    `cloudflared --version`

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Internet publica → SoundGrabber | Quick Tunnel torna a aplicação acessível por URL pública temporária |
| Operador → notebook | Instalação de `cloudflared` e inspeção de `.env` exigem ação humana com sudo |
| Cloudflare edge → localhost:8000 | Cloudflare faz proxy para a aplicação local sem abrir porta no roteador |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-15-01 | Information Disclosure | `/yonkou` público durante tunnel | mitigate | Gate bloqueante exige `ADMIN_PASSWORD` e `ADMIN_SESSION_SECRET` fortes antes de iniciar tunnel |
| T-15-02 | Spoofing | URL pública copiada incorretamente | mitigate | Acceptance exige URL `https://*.trycloudflare.com`; Plan 02 valida `/health` público |
| T-15-03 | Denial of Service | Exposição pública antes da origem saudável | mitigate | Task 1 bloqueia até Compose e `/health` local estarem OK |
| T-15-04 | Tampering | Instalação de binário externo | mitigate | Instalar via repositório APT oficial da Cloudflare e validar `cloudflared --version` |
</threat_model>
