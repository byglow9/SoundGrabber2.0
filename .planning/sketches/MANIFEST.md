# Sketch Manifest

## Design Direction
Visual Y2K/NFO autêntico — preto (#000) + laranja (#ff8800). Fontes bitmap (Sligoil + Dela Gothic One). Sem frameworks, sem CSS moderno no produto final. A direção de identidade está travada; o que exploramos agora é o "depth" visual — como dar peso e presença 3D aos painéis sem quebrar a autenticidade.

## Reference Points
- NFO BeatMasters International 2000 (referência original do projeto)
- phpBB / Orkut / Tibia era 2000–2005
- Imagens de referência do usuário: vibe 3D/isométrica, profundidade por camadas
