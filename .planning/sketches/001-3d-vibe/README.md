# Sketch 001: 3D Vibe

## Design Question
Qual dos 3 tratamentos 3D combina melhor com a estética NFO/Y2K (preto + laranja) sem quebrar a autenticidade?

## How to View
```
open .planning/sketches/001-3d-vibe/index.html
```

## Variants
- **A: Perspectiva 3D** — Painel inclinado com CSS `perspective` + `rotateX/Y`. Hover achata. Sombra dura laranja. Efeito "tela 3D flutuante". Usa CSS3.
- **B: Bevel Win98** — Bordas bicolores (claro top/left, escuro bottom/right). Zero CSS3. 100% CSS2. Estética Win98/2000 autêntica. Botões com press effect.
- **C: Bloco Extrudado** — Painel com camadas offset empilhadas atrás, criando "espessura". Header laranja sólido invertido. Botão de download com shadow dura.

## What to Look For
- A variante parece **autêntica Y2K** ou parece moderna demais?
- O efeito 3D **lê bem** ou distrai da usabilidade?
- Qual tem a melhor **"vibe" de ferramenta underground** pra produtores?
- A variante B (CSS2 puro) tem impacto visual suficiente?
